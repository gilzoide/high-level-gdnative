Do this:

#define HGDN_IMPLEMENTATION

before you include this file in one C or C++ file to create the implementation.

i.e.:

#include ...
#include ...
#define HGDN_IMPLEMENTATION
#include "hgdn.h"

Optionally provide the following defines with your own implementations:

  • HGDN_STATIC: If defined and HGDN_DECL is not defined, functions will be declared static instead of extern
  • HGDN_DECL: Function declaration prefix (default: extern or static depending on HGDN_STATIC)
  • HGDN_STRING_FORMAT_BUFFER_SIZE: Size of the global char buffer used for hgdn_print* functions. Defaults to 1024
